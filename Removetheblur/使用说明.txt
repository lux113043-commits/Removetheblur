========================================
图片背景修复工具 - 使用说明
========================================

【快速开始】

1. 安装依赖
   - 双击运行 install.bat
   - 或手动执行: pip install -r requirements.txt

2. 配置API密钥
   - 复制 env_example.txt 为 .env
   - 在 .env 文件中填入你的 OpenAI API Key

3. 启动Web服务
   - 双击运行 run_web.bat
   - 或手动执行: python web_app.py

4. 打开浏览器
   - 访问: http://localhost:5000

【使用步骤】

1. 在网页中输入图片文件夹路径
   例如: D:\Photos\input_images

2. 点击"开始处理"按钮

3. 等待处理完成（每张图片约20-60秒）

4. 查看处理结果对比

【输出位置】

处理后的图片保存在: 输入文件夹/fixed_images/

【注意事项】

- 需要有效的OpenAI API Key
- 需要稳定的网络连接
- 处理时间取决于图片数量
- 输出尺寸固定为 1024×1536

【技术支持】

如遇问题，请检查：
1. 依赖是否安装完整
2. API密钥是否正确
3. 网络连接是否正常
4. 文件夹路径是否正确
